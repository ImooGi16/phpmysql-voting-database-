# phpmysql-voting-database-

In order to use the phpmysql voting system, follow the steps below:

1) extract vsdb.rar to htdocs folder in your xampp folder

2) extract bower_components.rar inside the newly extracted vsdb folder

3) go to phpmyadmin, create a database named 'votingdatabase'

4) in the database named 'votingdatabase', import the sql file found in database folder

5) default admin login credentials are located in the 'read me first' txt file
